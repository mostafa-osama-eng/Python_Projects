# Employee System

## Description
A simple Python project to manage employees. 
Supports:
- Adding new employees
- Listing employees
- Deleting employees by age
- Updating employee salaries

## Features
- Object-oriented design with classes:
  - Employee
  - EmployeeManager
  - Frontend
- Input validation for menu choices
- Safe deletion of employees
- Clean separation between logic and I/O

Clone the repository.
